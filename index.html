FidgetDesk

FidgetDesk is a browser-based interactive experience that simulates a digital desktop filled with playful, physics-inspired objects. Users can click, drag, toss, and interact with items in real time, creating a satisfying and engaging environment designed for both entertainment and stress relief.

FidgetDesk was created to:

  • Provide a quick mental break during work or study sessions
  • Demonstrate interactive front-end development skills using HTML, CSS, and JavaScript
  • Explore event-driven programming and animation techniques in a fun, user-focused way
  • Deliver a lightweight, accessible experience that runs on both desktop and mobile browsers
