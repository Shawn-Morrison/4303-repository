# FidgetDesk

## Interactive Desk Toy Experience

### Description:

FidgetDesk is a browser-based interactive experience that simulates a digital desktop filled with playful, physics-inspired objects where users can click, drag, toss, and interact with items in real time, creating a satisfying and engaging environment designed for both entertainment and stress relief.

### Purpose and Audience:

FidgetDesk is a browser-based interactive desk designed to provide a quick mental break during periods of waiting or low engagement, such as being on hold or sitting in meetings. It was built for students, professionals, and casual users who want a simple, accessible way to stay lightly engaged without distractions.

### Tools and Technology:

Version Control \& Repository: GitHub - I chose this because of it's popularity and familiarity.
Desktop Git Interface: GitHub Desktop - I was familiar with utilizing this tool to publish versions.
Hosting Platform: GitHub Pages - This tools was chosen for familiarity, and mostly due to cost: *Free*
Languages: HTML, CSS, JavaScript built with Codex, review by a human before submission. In P01 I was handwriting this code or copying and pasting. 
AI Assistance: ChatGPT used for guidance, debugging, and learning support - This tool was chosen because familiarity and low cost. I recently adopted using CODEX by openAI because of it's more robust capabilities.

### From P01 to P02:

P01 had some issues with aesthetics, the lamp specifically did not look like a lamp. I fixed the lamp, added more pages to the notepad (based on feedback), improved/fixed a few physics elements, and added sound with a toggle to turn the sound on or off if desired.
